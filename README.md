Business Sales Performance Analysis
This repository contains a Business Intelligence (BI) project that transforms raw sales data into actionable insights. Using Python, Pandas, and Matplotlib, the analysis identifies top revenue-generating products and evaluates the effectiveness of store-level marketing strategies.

Inventory Value: Aggregated total price data by Material to understand the value distribution of the current stock.
